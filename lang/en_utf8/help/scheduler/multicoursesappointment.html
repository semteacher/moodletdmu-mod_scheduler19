Setting appointments slots overlapping within multiple courses

Teacher can customize a slot overlapping behaviour mode :

  • "No" - default setting: Slots overlapping never allowed.
  • "Yes": Slots overlapping are allowed only if overlapped slots are appointed in the scheduler models that are related to the different courses. Teacher can create overlapped slots in different courses. This mode is useful when teacher want to create schedule of the students missed class. Note. Existing exclusive or limited slots still has not been overlapped never.
