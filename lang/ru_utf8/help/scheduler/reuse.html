Расписание

Повтороне использование интервала

"Да" - "Использовать повторно" Интервал будет оставаться в расписании пока студент или преподаватель не отменят данное расписание приёма вцелом. Освобождённый интервал будет снова доступен для записи.

"Нет" - "Повторно не использовать" Интервал будет автоматически удалён при достижении момента окончания соответствующей даты (однако, предвидится, что Вы не принуждаете к закрытию "прямо сейчас"). Охранная задержка может быть установлена с помощью параметра "Охранная задержка для повторного использования".
